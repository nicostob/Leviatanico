# Leviatanico
La repository contiene tutti i file necessari all'esame di DH
Il progetto d'esame consiste nella creazione di un sito web (leviatanico.it) per la dimostrazione dell'utilità d'impiego delle tecniche e metodologie proprie delle Digital Humanities, dimostrando al contempo di averne compreso le questioni principali, e facendone propri i concetti fondanti.
L'intento del sito è quello di fornire una professionale raccolta e suddivisione catalogica, metadatazione e consultazione delle opere di Thomas Hobbes, valorizzandone esegeticamente i nodi ermeneutici, i concetti filosofici e le informazioni storico-biografiche. Il sito è da intendersi come una proposta di progetto ulteriormente implementabile.
Pertanto, tra i file principali, oltre a quattro pagine .html si annoverano file di dataset degli item, di metadatazione ed analisi in XML (con DC, DCTERMS e TEI), nonché immagini necessarie al layout del sito.
Le funzionalità principali di leviatanico.it riguardano la navigazione nelle quattro schede principali, un intuitivo e tematicamente coerente accesso al catalogo degli item, alla pagina dei dati e alla scheda di un item.
